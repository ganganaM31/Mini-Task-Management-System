# Task Management System

A simple Task Management System with frontend and backend functionality.

## Features
- Add tasks
- View tasks
- Update task status
- Delete tasks
- Search tasks

## Run locally

1. Open a terminal in `c:\Users\kalin\Desktop\Intern`
2. Install dependencies:

```bash
npm install
```

3. Start the server:

```bash
npm start
```

4. Open your browser at `http://localhost:3000`
